# Task 1 Intent

**Title:** Identify Current Hydrogen Version Running in Production

**Date:** 2026-06-04

**Mode:** INVESTIGATION ONLY — no code changes

## User Intent

identify what version of the Hydrogen (Remix-based React SSR) frontend is currently
running in production on Shopify — without requiring administrative access to the Shopify
dashboard.

## Goal

Determine, from local code artifacts alone, which version of the Hydrogen frontend is
"live" — i.e., the most recently built/deployed version.

## Out of Scope

- No code changes
- No deployments
- No Shopify admin lookups
